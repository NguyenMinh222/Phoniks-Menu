# PHONIKS QR Menu — fixed v2

Исправления:
- Поиск теперь можно закрыть кнопкой `×`, после этого снова доступны языки и категории.
- Убраны резкие пересчёты высоты и `transition: all`, поэтому шапка работает плавнее.
- Поиск рендерится с задержкой 80 мс, чтобы не лагал при вводе.
- Меню рендерится порциями, а не одним большим блоком.
- Первые изображения загружаются приоритетно, остальные — lazy loading.
- Добавлен skeleton/fade-in для фото.
- Service Worker обновлён до `phoniks-menu-v4`.
- `menu-data.js` не кэшируется жёстко, чтобы правки в меню быстрее доходили до гостей.
- Картинки кэшируются после первой загрузки, чтобы повторные открытия были быстрее.
- Поддержаны двойные категории через `categories: ["soups", "vegan"]`.
- Добавлены категории `vegan` и `sets`.

## Как закрыть поиск
Нажать кнопку `×` справа в строке поиска или клавишу Escape на компьютере.

## Двойная категория блюда
Можно писать так:

```js
categories: ["soups", "vegan"],
category: "soups",
```

Поле `category` оставлено для совместимости, но фильтрация смотрит на `categories`, если оно есть.
