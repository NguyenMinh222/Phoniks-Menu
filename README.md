# PHONIKS QR Menu

## Как добавлять PNG-фото

1. Положи файл в папку `images`.
2. Назови файл латиницей без пробелов:
   - правильно: `pho-bo.png`
   - неправильно: `Фо Бо.png`
3. В `menu-data.js` пропиши:

```js
image: "images/pho-bo.png"
```

Код поддерживает PNG напрямую. Ничего конвертировать не нужно.

## Где редактировать меню

Все блюда находятся в `menu-data.js`.

## Бесплатная публикация без GitLab

Лучше варианты:
- GitHub Pages
- Netlify Drop
- Cloudflare Pages

Для самого простого запуска: Netlify Drop → перетащи всю папку проекта → получишь ссылку → сделай QR.
