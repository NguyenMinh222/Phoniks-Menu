# PHONIKS QR Menu — финальная версия

## Что добавлено

1. Категории `Веганское` и `Сеты для двоих`.
2. Одно блюдо может быть в нескольких категориях:

```js
category: ["soups", "vegan"]
```

3. При переключении категории сайт прокручивает меню к началу.
4. При прокрутке скрываются лишние элементы шапки: логотип и поиск, остаются языки и категории.
5. Поиск сворачивается в кнопку 🔎 слева от категорий.
6. Если пользователь работает с поиском и скроллит, сверху остаётся только поиск.
7. Поиск корректно обрабатывает много пробелов.
8. При новом открытии QR-меню стартует с русского языка и категории «Все».
9. Для блюд без фото появляется подпись: «фото блюда скоро появится».
10. Service Worker обновлён до `phoniks-menu-v4`, чтобы меньше залипали старые версии.

## Как добавить блюдо в две категории

```js
{
  id: "pho-dau",
  category: ["soups", "vegan"],
  ...
}
```

## Как добавить сет для двоих

```js
{
  id: "beer-set",
  category: ["snacks", "sets"],
  ...
}
```

## Как добавить фото

Файл кладёшь в папку `images`, а в блюде указываешь:

```js
image: "images/pho-bo.webp"
```

Если фото пока нет:

```js
image: ""
```

Сайт сам покажет заглушку и подпись.
