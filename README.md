# PHONIKS QR Menu — stable v3

Исправления:
- карточки блюд возвращены к прежнему размеру;
- убраны тяжёлые анимации карточек и skeleton-анимация;
- поиск закрывается стабильно через кнопку `×`;
- после закрытия поиска интерфейс не зацикливается и не дёргается;
- категории снова доступны сразу после закрытия поиска;
- кэш обновлён до новой версии.

Фото по-прежнему кладутся в папку `images`.


## Версия v4 search/scroll fix

- Поиск больше не учитывает название категории, поэтому не показывает лишние блюда.
- Поиск разбивает запрос на слова: блюдо показывается только если найдены все слова запроса.
- Быстрый скролл стабилизирован: шапка больше не пересчитывается на каждом пикселе прокрутки.
- Кэш обновлён до `phoniks-menu-v5`.
