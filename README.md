# PHONIKS QR Menu

## Как добавлять PNG-фото

1. Положи файл в папку `images`.
2. Назови файл латиницей без пробелов:
   - правильно: `pho-bo.png`
   - неправильно: `Фо Бо.png`
3. В `menu-data.js` пропиши:

```js
image: "images/pho-bo.png"
```

Код поддерживает PNG напрямую. Ничего конвертировать не нужно.

## Где редактировать меню

Все блюда находятся в `menu-data.js`.

## Бесплатная публикация без GitLab

Лучше варианты:
- GitHub Pages
- Netlify Drop
- Cloudflare Pages

Для самого простого запуска: Netlify Drop → перетащи всю папку проекта → получишь ссылку → сделай QR.


## Граммовки

В каждом блюде есть поле:

```js
weight: "350 г"
```

Можно писать так:

```js
weight: "450 г + рис 120 г"
```

Если граммовка пока неизвестна, оставь:

```js
weight: "уточняется"
```

## Баннер обеденного меню

Баннер уже добавлен в `index.html`. Просто положи свою картинку сюда:

```text
images/lunch-banner.png
```

Если картинки нет, баннер автоматически скрывается и не ломает сайт.

## Перевод граммовок

Можно оставлять простой вариант:

```js
weight: "450 г + рис 120 г"
```

Сайт автоматически переведёт единицы:
- `г` → `g` / `克`
- `мл` → `ml` / `毫升`
- `рис` → `rice` / `米饭`

Для самого точного варианта можно писать так:

```js
weight: {
  ru: "450 г + рис 120 г",
  en: "450 g + rice 120 g",
  vn: "450 g + cơm 120 g",
  zh: "450克 + 米饭120克"
}
```

Если стоит:

```js
weight: "уточняется"
```

то сайт покажет:
- RU: граммовка уточняется
- EN: weight to be confirmed
- VN: khối lượng đang cập nhật
- ZH: 重量待确认
